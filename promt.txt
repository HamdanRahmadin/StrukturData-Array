Saya adalah mahasiswa Informatika dan mendapatkan tugas Struktur Data.
Buatkan program menggunakan struktur data array statis (fixed-size array) 
dengan kapasitas 10 untuk menyimpan data mahasiswa berupa NIM dan Nama.

Program harus dibuat dalam dua bahasa:
1. Java
2. Python

Ketentuan teknis:
- Gunakan array statis, bukan dynamic array
- Tidak boleh menggunakan library untuk operasi array
  (contoh: ArrayList, list.remove, list.insert, dsb)
- Gunakan variabel `count` untuk menandai jumlah data aktif di array
- Setiap operasi insert: count bertambah 1
- Setiap operasi delete: count berkurang 1
- Operasi geser data harus dilakukan secara manual menggunakan perulangan

Program harus memiliki menu:
1. Insert at beginning
2. Insert at given position
3. Insert at end
4. Delete from beginning
5. Delete given position
6. Delete from end
7. Delete first occurrence (berdasarkan NIM)
8. Show data
9. Exit

Ketentuan tambahan:
- Lakukan validasi jika array penuh saat insert
- Lakukan validasi jika array kosong saat delete
- Lakukan validasi posisi agar tidak out of bounds
- Program berjalan menggunakan input menu (console)

Mohon sertakan:
- Source code yang rapi dan terstruktur
- Komentar penjelasan pada setiap bagian penting program
- Penjelasan singkat logika array dan penggunaan variabel count
